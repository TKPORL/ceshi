# Tsinho 个人主页

这是一个多页面、可交互的个人主页网站。

## 快速部署到 GitHub Pages

### 方法一：使用 Git 命令行（推荐）

```bash
# 1. 克隆仓库
git clone https://github.com/TKPORL/ceshi.git
cd ceshi

# 2. 添加远程仓库（如果需要）
git remote add origin https://github.com/TKPORL/ceshi.git

# 3. 推送到 GitHub
git push -u origin master
```

### 方法二：直接上传文件

将以下文件上传到 GitHub 仓库：
- `index.html` - 首页
- `portfolio.html` - 作品集页面
- `about.html` - 关于页面
- `contact.html` - 联系页面
- `style.css` - 样式文件
- `script.js` - JavaScript 交互文件

### 启用 GitHub Pages

1. 进入 GitHub 仓库设置
2. 找到 "Pages" 部分
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "master" 和 "/(root)"
5. 点击 Save

等待几分钟后，你的网站将在 `https://tkporl.github.io/ceshi` 可访问。

## 网站功能

- 响应式设计，适配各种设备
- 作品集展示（支持分类筛选）
- 技能展示（动画进度条）
- 工作经历时间线
- 联系表单（调用邮件客户端）
- 平滑滚动和过渡动画
